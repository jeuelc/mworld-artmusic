What's Missing

By now you should have a range of drawings, some using line, some using four tones, some using hatching, some using a gradation of tone. These are what we call observational drawings. Notice any improvements as you have gone along?

If you have done each drawing as suggested, you will be able to see that with effort and persistence you can get much better at this and begin to make some wonderful drawings.

Once youve reached this point we can begin to ask some more questions about our approach to drawing. If what you are doing is trying to represent the objects as completely as possible, then you might ask: what is missing from your attempt to capture the real?
